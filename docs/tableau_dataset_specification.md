## Purpose
This document is the authoritative field-binding and data specification for `EV_Final_Ready_Tableau.csv`.   - `Not eligible due to low battery range`

- **Cleaning**: No changes applied to source column

---

### Column 11 — Electric Range
- **Tableau Field Name**: `Electric Range`
- **Type**: Float
- **Description**: EPA-rated electric-only driving range in miles — primary analysis column
- **Range (valid values)**: 1.0 – 337.0 miles
- **Mean (NaN-excluded)**: ~107.5 mi
- **Nulls after cleaning**: Present — original zeros replaced with `NaN`
- **Cleaning**: All `Electric Range = 0` values replaced with `np.nan`. Zero in the DOL registry means "range not researched" — not a literal zero-mile vehicle. Original values preserved in `Electric Range Raw`.
- **Impact**: 179,538 zeros → NaN
- **Dashboard use**: Average range KPI card, range evolution line chart, scatter plot Y-axis. Do NOT use for population count charts.

### Column 12 — Legislative District

- **Nulls**: Present → filled with `'Unknown'`

### Column 13 — DOL Vehicle ID

### Column 14 — Vehicle Location
- **Tableau Field Name**: `Vehicle Location`

- **Nulls**: Present in raw → filled with `'Unknown'`

- **Dashboard use**: Source field only — use `Latitude` and `Longitude` columns for all maps

### Column 17 — Electric Range Raw
- **Tableau Field Name**: `Electric Range Raw`
- **Type**: Float
- **Description**: Exact copy of the original `Electric Range` column, made **before** zero → NaN substitution. Includes all zeros.
- **Purpose**: Used for `Range_Category` binning and population count charts where zero = "not researched" (not excluded)
- **Cleaning**: `df['Electric Range Raw'] = df['Electric Range'].copy()` — executed before NaN treatment
- **Dashboard use**: Source for `Range_Category`; population volume counts. Do NOT use for average range KPIs.

## Calculated Fields — Tableau Definitions

Create these in Tableau via **Analysis → Create Calculated Field**:

```
[Total Registrations]
COUNT([DOL Vehicle ID])

[BEV Share %]
COUNTD(IF [EV Type] = "Battery Electric Vehicle (BEV)" THEN [DOL Vehicle ID] END)
/ COUNT([DOL Vehicle ID])

[PHEV Share %]
COUNTD(IF [EV Type] = "Plug-in Hybrid Electric Vehicle (PHEV)" THEN [DOL Vehicle ID] END)
/ COUNT([DOL Vehicle ID])

[Avg Electric Range]
AVG([Electric Range])

[CAFV Eligible Rate]
COUNTD(IF [Eligibility Status] = "Eligible" THEN [DOL Vehicle ID] END)
/ COUNT([DOL Vehicle ID])

[Luxury Share %]
COUNTD(IF [Market Segment] = "Luxury" THEN [DOL Vehicle ID] END)
/ COUNT([DOL Vehicle ID])

[Mass Adoption Count]
COUNTD(IF [Adoption Wave] = "Mass Adoption" THEN [DOL Vehicle ID] END)

[Tesla Share %]
COUNTD(IF [Make] = "TESLA" THEN [DOL Vehicle ID] END)
/ COUNT([DOL Vehicle ID])
```

## Known Data Anomalies & Handling Notes

| Anomaly | Scale | Handling |
|---|---|---|
| `Electric Range = 0` ("not researched") | 179,538 rows (64% of raw) | Replaced with `NaN` in `Electric Range`; preserved in `Electric Range Raw` |
| Postal Code as float (`98034.0`) | All rows | Converted to integer string |
| Missing County / City / Legislative District | <1% of rows | Filled with `'Unknown'` |
| Corrupt / missing POINT strings | ~few hundred rows | Dropped — required for map integrity |
| Lat/Long outliers outside WA State | ~small number | Likely DOL data entry errors; not filtered |
| PHEV range under-representation | Systemic | PHEVs disproportionately lack range data → analytical subset skews BEV-heavy (68.7% vs 45% in full data) |
| CAFV eligibility verbose text | All rows | Simplified to `Eligibility_Status`; source column retained |
